# Tech Stack

- Python 3.10
- Required Libaries
    - psycopg2
- Postgres Server
- pgAdmin 4

# Setup Guide

- Install python
- Install libraries
    - pip3 install psycopg2
- Create a postgres DB
- Run the DDL queries to configure the DB

# Run Instructions

- Navigate to 2.5-Implementation
- Command:
    - macOS / Linux: python3 main.py
    - Windows: python3 main.py
